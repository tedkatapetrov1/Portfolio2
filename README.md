This is a simple, responsive portfolio website built using **HTML**, **CSS**, **JavaScript**, and **Bootstrap**.  
It features multiple pages including:

- 🏠 Home  
- 👨‍💼 About  
- 💻 Experience  
- 🎓 Education  
- 📄 Certificates  
- ✉️ Contact
- 💡 Projects
